Project Overview

Sistem monitoring dan proteksi motor DC berbasis IoT dengan integrasi Edge Machine Learning pada ESP32. Sistem menggunakan pendekatan closed-loop control untuk mendeteksi dan mencegah overheating secara otomatis.
